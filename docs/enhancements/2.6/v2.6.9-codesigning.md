# v2.6.9 — Codesigning with Sigstore

## Overview

Sign all release artifacts using Sigstore keyless signing to provide supply chain integrity verification. Uses GitHub Actions OIDC tokens — no private keys to manage. Signatures are recorded in Rekor transparency log for public verifiability.

## Prerequisites

- Phases 1–8 complete (all code compiled and tested)
- GitHub repository configured with Actions enabled

## Files to Create

| File | Action |
|------|--------|
| `rust-nexus/.github/workflows/release.yml` | CI pipeline: build, sign, verify, release |
| `rust-nexus/scripts/verify-release.sh` | End-user signature verification script |

## Implementation Details

### 9.1 GitHub Actions Release Workflow

```yaml
# .github/workflows/release.yml
name: Build, Sign, and Release
on:
  push:
    tags: ['v*']

jobs:
  build-and-sign:
    runs-on: ubuntu-latest
    permissions:
      contents: write      # Create releases
      id-token: write      # Sigstore OIDC keyless signing
      packages: write      # Push container images

    steps:
      - uses: actions/checkout@v4
        with:
          persist-credentials: false

      - name: Install Cosign
        uses: sigstore/cosign-installer@v4.0.0

      - name: Install Rust toolchain
        uses: dtolnay/rust-toolchain@stable
        with:
          targets: x86_64-unknown-linux-gnu,x86_64-pc-windows-gnu

      - name: Install cross-compilation tools
        run: sudo apt-get install -y gcc-mingw-w64-x86-64

      # ── Build ───────────────────────────────────────────────
      - name: Build Linux binaries
        run: |
          cargo build --release -p nexus-agent --target x86_64-unknown-linux-gnu
          cargo build --release -p nexus-sandbox --target x86_64-unknown-linux-gnu

      - name: Build Windows binaries
        run: |
          cargo build --release -p nexus-agent --target x86_64-pc-windows-gnu
          cargo build --release -p nexus-sandbox --target x86_64-pc-windows-gnu

      # ── Sign Binaries ──────────────────────────────────────
      - name: Sign release binaries (keyless)
        run: |
          for target in x86_64-unknown-linux-gnu x86_64-pc-windows-gnu; do
            for bin in nexus-agent nexus-sandbox; do
              artifact="target/${target}/release/${bin}"
              [ -f "${artifact}.exe" ] && artifact="${artifact}.exe"
              if [ -f "$artifact" ]; then
                cosign sign-blob "$artifact" \
                  --bundle "${artifact}.sigstore.json" --yes
                echo "Signed: $artifact"
              fi
            done
          done

      # ── Sign Proto Files ───────────────────────────────────
      - name: Sign protocol definitions
        run: |
          for proto in nexus-infra/proto/*.proto; do
            cosign sign-blob "$proto" \
              --bundle "${proto}.sigstore.json" --yes
            echo "Signed: $proto"
          done

      # ── Verify All Signatures ──────────────────────────────
      - name: Verify all signatures
        run: |
          IDENTITY="https://github.com/${{ github.repository }}/.github/workflows/release.yml@${{ github.ref }}"
          ISSUER="https://token.actions.githubusercontent.com"

          for bundle in $(find . -name "*.sigstore.json"); do
            artifact="${bundle%.sigstore.json}"
            echo "Verifying: $artifact"
            cosign verify-blob "$artifact" \
              --bundle "$bundle" \
              --certificate-identity="$IDENTITY" \
              --certificate-oidc-issuer="$ISSUER"
          done

      # ── Container Image ────────────────────────────────────
      - name: Login to GHCR
        uses: docker/login-action@v3.6.0
        with:
          registry: ghcr.io
          username: ${{ github.actor }}
          password: ${{ secrets.GITHUB_TOKEN }}

      - name: Build and push container
        uses: docker/build-push-action@v6.18.0
        id: build-and-push
        with:
          push: true
          tags: ghcr.io/${{ github.repository }}/nexus-agent:${{ github.ref_name }}

      - name: Sign container image
        env:
          DIGEST: ${{ steps.build-and-push.outputs.digest }}
        run: |
          cosign sign --yes \
            "ghcr.io/${{ github.repository }}/nexus-agent:${{ github.ref_name }}@${DIGEST}"

      # ── Release ─────────────────────────────────────────────
      - name: Create GitHub Release
        uses: softprops/action-gh-release@v1
        with:
          generate_release_notes: true
          files: |
            target/x86_64-unknown-linux-gnu/release/nexus-agent
            target/x86_64-unknown-linux-gnu/release/nexus-agent.sigstore.json
            target/x86_64-unknown-linux-gnu/release/nexus-sandbox
            target/x86_64-unknown-linux-gnu/release/nexus-sandbox.sigstore.json
            target/x86_64-pc-windows-gnu/release/nexus-agent.exe
            target/x86_64-pc-windows-gnu/release/nexus-agent.exe.sigstore.json
            target/x86_64-pc-windows-gnu/release/nexus-sandbox.exe
            target/x86_64-pc-windows-gnu/release/nexus-sandbox.exe.sigstore.json
            nexus-infra/proto/nexus.proto
            nexus-infra/proto/nexus.proto.sigstore.json
            nexus-infra/proto/sandbox.proto
            nexus-infra/proto/sandbox.proto.sigstore.json
            scripts/verify-release.sh
```

### 9.2 End-User Verification Script

```bash
#!/bin/bash
# verify-release.sh — Verify Sigstore signatures on nexus release artifacts
set -euo pipefail

REPO="${REPO:-cmndcntrlcyber/rust-nexus}"
ISSUER="https://token.actions.githubusercontent.com"

echo "Verifying Sigstore signatures for rust-nexus release artifacts"
echo "Repository: ${REPO}"
echo ""

PASS=0
FAIL=0

for artifact in nexus-agent nexus-agent.exe nexus-sandbox nexus-sandbox.exe \
                nexus.proto sandbox.proto; do
  if [ -f "${artifact}" ] && [ -f "${artifact}.sigstore.json" ]; then
    echo -n "  ${artifact}: "
    if cosign verify-blob "${artifact}" \
        --bundle "${artifact}.sigstore.json" \
        --certificate-identity-regexp="https://github.com/${REPO}/.*" \
        --certificate-oidc-issuer="${ISSUER}" 2>/dev/null; then
      echo "VERIFIED"
      PASS=$((PASS + 1))
    else
      echo "FAILED"
      FAIL=$((FAIL + 1))
    fi
  fi
done

echo ""
echo "Results: ${PASS} verified, ${FAIL} failed"
[ "${FAIL}" -eq 0 ] && exit 0 || exit 1
```
