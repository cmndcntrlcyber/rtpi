Automated penetration test report generation from accumulated mem0 findings.

## Capabilities

- Executive summary from critical/high findings
- Technical narrative using attack path graph
- CVSS 3.1 scoring automation
- CWE and MITRE ATT&CK mapping
- Remediation priority matrices
- Evidence documentation and screenshot management
- Attack chain visualization from graph memory
